## Getting Started

1. Clone repo:

```bash
git clone https://github.com/your-username/promptly.git
cd promptly
```

2. Install dependencies:

```bash
pnpm install
```

3. Create `.env` file:

```env
OPENAI_API_KEY=your_openai_key_here
```

4. Run development server:

```bash
pnpm dev
```
